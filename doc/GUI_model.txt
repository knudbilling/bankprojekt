Model over GUI vinduer.

Første "side" er en "vælg brugertype":
+--------------------------------------------------------------------------+
|   						"BANKNAME" INTERFACE						   |
+--------------------------------------------------------------------------+
|																		   |
|  	Tast "1" hvis du er kunde.											   |
|	Tast "2" hvis du er medarbejder i banken.							   |
|																		   |
|	Tast "Q" for at afslutte.											   |
|																		   |
|	Godkend med "Enter".												   |
|																		   |
+--------------------------------------------------------------------------+

Hvis brugertypen er "kunde":
+--------------------------------------------------------------------------+
|   						"BANKNAME" INTERFACE						   |
+--------------------------------------------------------------------------+
|																		   |
|	Indtast kundenummer:												   |
|																		   |
|	Tast "B" for at gå tilbage.											   |
|	Tast "Q" for at afslutte.											   |
|																		   |
|	Godkend med "Enter".												   |
|																		   |
+--------------------------------------------------------------------------+
